# エアホッケー対戦ゲーム

2台のiPadで遊べるリアルタイム対戦エアホッケーゲームです。

## 遊び方

1. 2台のiPadをそれぞれ横向きにする
2. 片方のiPadで **「つくる」** を選ぶ（ホスト）
3. もう片方のiPadで同じURLを開き **「はいる」** を選ぶ（ゲスト）
4. ホスト画面に表示された **4桁の合言葉** をゲスト側に入力する
5. 接続したらゲームスタート！先に **5点** 取ったほうの勝ち

## ハンデ機能

年齢差がある場合、ホストの待機画面でハンデをオンにできます。  
「ちいさい子」側のゴールを広く・マレットを大きくして、対等に戦えます。

## 技術構成

- **単一HTMLファイル** — ビルド不要、`index.html` をそのまま配置するだけ
- **PeerJS 1.5.4** — WebRTC P2P通信（ホスト権威モデル）
- **Canvas API** — ゲーム描画（テーブル・パック・マレット）
- **バニラJS** — フレームワーク不使用

## ローカルで動かす

```bash
npx serve .
# → http://localhost:3000 を2つのタブ/デバイスで開く
```

## デプロイ

GitHub Pages などの静的ホスティングに `index.html` を置くだけで動作します。
